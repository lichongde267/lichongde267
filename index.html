注水

发表于 2021-01-14 | 更新于 2021-01-14 | 分类于 独白 | | 阅读次数:

对这个充满水分的世界充满不屑。
看着人们注水的身材，听着人们注水式地说话
好像我常看话剧，荒唐、夸张，他们努力逼自己演技
怎么世界就交给了这帮傻叉了呢？
这似乎好像你某天在街头见到热爱过的姑娘，曾经抱过的腰如柳的姑娘，瞬间就成邮政筒了。
然后自己转头思考：咦，青春要散场了吗？
注水牛肉，注水的肉体，胖。胖在，胖崽，在即合理？
但是注水是合理的吗？注水互害？

有时候你真的不怎么瞧得上这个世界
当然，我是世界的一员，同理得瞧不起自己。
（此推理不适用其他人）

铭记冒着睾丸癌危险用前列腺丈量大地的人。

《2020给自己的信》

发表于 2021-01-14 | 更新于 2021-01-14 | 分类于 日记 | | 阅读次数:

亲爱的李重德：
见信如晤
2020年，疫情爆发前，我提醒过你，要小心，尽管我提醒了你并安慰你也许世界会好的，可依然突破我们两个的预测，它以无法改变地速度改变了人类栖息。这两年，我陪伴你的时间少了很多，我惊奇地发现，你有时候很傻叉。

我不喜欢你，你大概也不会喜欢我。我无趣，凭什么要求你有趣？我都不优秀，凭什么要求你？尽管你不愿意听，首先我还是希望你道歉，大大的道歉，阿土说得对，你也许要对微信所有的亲朋好友，对那些给你发信息了你却没及时回的人，甚至直接没回的人道歉。我知道你的电子设备越来越多，是累赘和抗拒，但你还是应该回复下，至少出于礼貌要回。

传说：“你每天晚上睡觉都被自己折磨，夜里彷徨，对着深渊问自己：怎么办？”这是胡扯，我和你认识二十多年，你从来都是一觉儿到天明，半夜从来不起来撒尿。

面对未来：
你还不像一道光，有了你好像也一样，没了你其实也一样，你不过是具肉体，尘归尘土归土，还有几十年，只剩几十年，想想少年老成的脸如何能挂在墙上。你常常开车进黑暗，只有那时，你和发动机有光，它只能划破黑暗，一点点。做光还是画框？

距离：
我们之间从来好像从来没有那么接近过，几颗树的距离，从没有好好说过话，就好像你跟你家人亲人爸妈弟，从未好好说过话。有我陪你，寂寞什么？元件不是对丘奇说过：我没爱够你，不准你放弃，我想放弃的时候会跟你说。她说得对，我会像她那样爱你的。

声音：
少言，少言，少言，不想听到你的声音。
偶尔有什么话别出来。

远行：
没有人能跟你去远行，我也知道，但我也希望你尝试着，带姑娘去远行，试试。不要害怕朝向西边的生活，那里充满着无尽的阳光。

关于权钱色：
先说色，低级欲望，初级欲望，嫖娼可以解决？我知道你有色心，没色胆，不敢。又，你常常把姑娘当艺术品？很傻，你会失去泡姑娘的能力的，书和电子机器真比姑娘有趣吗？我看未必，你居然可以把好看的姑娘当你去博物馆、艺术馆看古器物、艺术作品那样看待她们？为什么呢？
权，它能干嘛？它可以做事，但是你不会想向林奇那样被投毒吧？千军万马也会倒的，马会死的，马累死是有可能的。
尤其资本主义的旋律之下的所谓工作八小时的效率，为的是让工人8小时竭尽所能成为流水线的机器。
钱，钱不是生活质量决定性因素，也不是生命纬度重要的东西。小时候我常看香港电影，一个主旋律就是黑帮大佬赚够之后给自己洗白退休，想退休？没完，必须带着兄弟们继续赚，但是兄弟们没够，退休然后被兄弟们冲进家里一枪嘣了。

侵袭

发表于 2021-01-14 | 更新于 2021-01-14 | 分类于 日记 | | 阅读次数:

2020一场突如其来的疼痛，让我更加珍惜生命，让我知道，人类还可以野蛮生长。这场暴风雪搬的、惨绝人寰的疼痛将近在我的右口腔，并持续无比长的5分钟。它反复发作，用蛮荒之力，久久不能根除，在短时间内，甚至差点让我用意念圆寂了自己。

这场突如其来的牙痛让我意识到，生命宝贵，人间凶险。
阳光猛烈，动物凶猛，它一样不听话地长在脖子上的脑袋，2021年，发誓斩草除根！
